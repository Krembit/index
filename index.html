Key Terms

  • World Wide Web

    • This refers to the links between webpage files. This is what your computer shows you when you open your internet browser.

  • Internet

    • This refers to the series of servers and storage the devices that house the files that make up the web, and the wired and wireless connections between these servers.

  • Browser

    • A browser is a computer program that is able to download the files that make up the World Wide Web by contacting the server holding the file via the Internet. Browsers also make it easier to find new web pages.
